# 내 개발 포트폴리오

## 페이지 구성 (컴포넌트)

1. About me
   - 프로젝트 목록(클릭 시 자동 스크롤)
   - 스킬 목록
2. Projects
   - 프로젝트 명
   - 한 줄 설명
   - 프로젝트 시연 영상 (별도의 유튜브 영상 가져오기)
   - 프로젝트 핵심기능 코드 일부 ‘코드 블록’에 삽입하여 보여주기
   - 상세 정보가 담긴 깃허브 프로젝트 링크
3. Contact
   - 이메일(복사버튼)
   - 번호(복사버튼)
   - 거주지(카카오맵으로 주소만 보이게 + 확대 축소만 가능하게)
